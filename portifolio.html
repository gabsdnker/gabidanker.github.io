Assistente de Estudos com Agenda Inteligente

Este projeto é um sistema de gerenciamento de tarefas e projetos voltado para estudantes, que oferece uma interface gráfica simples e sugestões inteligentes baseadas em horários. A aplicação foi desenvolvida em Java e utiliza 6 Padrões de Projeto como parte da arquitetura.


This project is a task and project management system aimed at students, offering a simple graphical interface and intelligent suggestions based on schedules. The application was developed in Java and uses 6 Design Patterns as part of its architecture.

Este projeto é um sistema de gerenciamento de tickets para eventos, permitindo que os usuários comprem, vendam e gerenciem seus ingressos de forma eficiente. A aplicação foi desenvolvida em Python e utiliza o framework Django.


This project is a ticket management system for events, allowing users to buy, sell, and manage their tickets efficiently. The application was developed in Python and uses the Django framework.
